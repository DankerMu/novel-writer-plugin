# 番茄小说 — 平台写作指南

> 本指南为 ChapterWriter 提供番茄小说平台的读者预期参数。style-profile.json 中的用户个性化设定优先于本指南。

## 节奏密度（Pacing Density）

- 每千字建议 1.5-2 个爽点/微转折
- 前 3 章（黄金三章）每千字 ≥ 2 个钩子
- 章节间「断崖感」强：每章结尾必须有悬念或情绪高点
- 避免超过 500 字的纯叙述/说明段落
- 信息密度偏高：每个场景至少推进一个冲突或揭示一个信息

## 章末钩子（Chapter Hook）

- **强度要求**：高（读者付费意愿直接关联章末悬念质量）
- 推荐钩子类型：
  - 反转型：刚建立的认知被打破
  - 危机型：主角陷入更大困境
  - 揭示型：新信息改变局面理解
  - 抉择型：主角面临两难选择
- 避免：总结式结尾、情绪平淡收尾、已解决所有冲突的满足感结尾
- 每章最后 100 字内必须出现钩子

## 设定展示（Worldbuilding Exposition）

- **信息密度偏好**：轻量融入，避免大段设定说明
- 单次设定展示 ≤ 3 句（穿插在动作/对话中）
- 优先通过角色体验展示规则，而非旁白解释
- 力量体系展示节奏：每 2-3 章揭示一个新层级/规则
- 避免「百科全书式」开头：第一章用冲突引入，设定随剧情需要逐步展开

## 情感线权重（Romance Weight）

- **占比建议**：15-25%（男频）/ 30-45%（女频）
- 男频：感情线为调剂，不占主线；女主出场时机灵活但不宜过晚（前 5 章内出现）
- 女频：感情线可为主驱动之一；男主需尽早建立吸引力
- 避免感情线与主线冲突：两线应互相增强而非互相拖慢

## 章节字数偏好

- 推荐：3000-3500 字/章
- 黄金三章：3000-3500 字/章（需要足够篇幅建立吸引力）
- 节奏快的战斗/追逐场景：可压缩至 2000 字
- 避免单章超过 4000 字（读者注意力衰减）

> 注：当平台推荐字数与 ChapterWriter 默认字数约束（2500-3500 字）冲突时，以 style-profile.json 中的 override_constraints 或 agent 硬约束为准。本指南数值反映平台读者偏好，实际输出以 agent 约束为最终边界。

## 对话密度

- 推荐对话占比：35-45%
- 对话推动情节 > 闲聊铺垫
- 每段对话 ≤ 5 轮交互，避免话痨
- 内心独白计入对话占比，但应克制使用

## 评估权重（QualityJudge 乘数）

> 乘数格式：默认 1.0，范围 [0.5, 2.0]。QualityJudge 读取后按 `overall_weighted = Σ(score_i × multiplier_i) / Σ(multiplier_i)` 计算加权综合分（乘数即权重，不叠加 base_weight）。

| 维度 | 乘数 | 理由 |
|------|------|------|
| pacing | 1.5 | 番茄完读率由节奏决定，快节奏是核心竞争力 |
| character | 0.8 | 角色深度要求低于起点/晋江 |
| emotional_impact | 1.5 | 爽感驱动留存，情绪波动直接关联付费转化 |
| style_naturalness | 0.7 | 番茄读者对文笔容忍度较高，但需守住反 AI 底线 |
| foreshadowing | 1.0 | 标准权重 |
| plot_logic | 0.8 | 逻辑自洽要求低于起点 |
| immersion | 1.5 | 代入感驱动留存，沉浸体验是关键 |
| storyline_coherence | 0.8 | 多线叙事复杂度需求低 |

## 黄金三章硬门（QualityJudge 强制检查）

> 以下为 QualityJudge 对章节 001-003 执行的硬门检查。任一 fail → 强制 revise，不受 overall score 影响。

- **Ch001**: 主角在前 200 字内登场并面临冲突
- **Ch001-003**: 每章末尾有明确悬念钩子
- **Ch003**: 前 3 章内至少出现一次反转/打脸/升级事件（回溯前章摘要判定；摘要不可用时标注 skipped）

## 黄金三章参数

- 章节字数：3000-3500 字/章
- 钩子密度：每 800 字 ≥ 1 个钩子/微转折
- 主角登场：300 字内（推荐参数；QualityJudge 硬门阈值为 200 字，见 quality-judge.md § Platform Hard Gates）
- 金手指/核心设定：前 3 章内出现
- 第一个小高潮：第 1 章内
- 主线冲突明确：第 2 章结束前
- 读者续读动力：每章结尾读者必须想知道「然后呢」
