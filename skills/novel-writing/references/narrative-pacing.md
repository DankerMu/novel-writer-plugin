# 叙事节奏设计方法论

本文档为 PlotArchitect 规划卷级大纲时提供叙事节奏设计指导。核心目标：让读者**追着看**，而非每章独立达标但整体平淡。

## 使用方式

PlotArchitect 在设计本卷章节结构（Process 步骤 3）时：
1. 根据卷总章数选择卷级弧线模板（§1）
2. 在弧线框架内划分 Phase，为每章赋值 `- **Phase**:` key 行
3. 在 Phase 内安排蓄力→爆发→收束小循环（§2）
4. 根据 Phase + 小循环位置选择 excitement_type（§3）
5. 编排钩子链节奏（§4）
6. 布局信息差与承诺-兑现周期（§5, §6）

> 本文档为方法论指导，PlotArchitect 可根据具体项目 genre 和用户方向指示灵活调整——模板是骨架，不是枷锁。

## §1 卷级弧线模板（Phase 体系）

### 1.1 五段式弧线（默认模板）

网文卷级结构的通用节奏，适用于绝大多数 genre：

| Phase | 名称 | 功能 | 占比 | 情绪走向 |
|-------|------|------|------|---------|
| `期待` | 建立期待 | 新地图/新势力/新目标引入，让读者产生"接下来会怎样" | ~15% | ↗ 上升 |
| `试探` | 初步试探 | 事态顺利发展，小胜利，读者感到"一切都好" | ~20% | ↗ 平稳上升 |
| `受挫` | 真正受挫 | 真正障碍出现，幻象破灭，主角暴露弱点 | ~25% | ↘ 下降 |
| `噩梦` | 至暗时刻 | 多线压力交汇，局面近乎绝望 | ~25% | ↘↘ 谷底 |
| `爆发` | 决定性爆发 | 主角克服内外困难，大爽点释放 + 卷末收束 | ~15% | ↗↗ 顶峰 |

**以 25 章卷为例**：

```
Ch 1-4   [期待]  新地图/势力引入，卷目标明确化
Ch 5-9   [试探]  初步试探成功，获得小资源，小打脸
Ch 10-15 [受挫]  真正的敌人登场，信息揭示主角劣势
Ch 16-21 [噩梦]  多线压力交汇，看似无解
Ch 22-25 [爆发]  逆转 + 大打脸 + 伏笔回收 + 卷末钩子
```

**以 30 章卷为例**：

```
Ch 1-5   [期待]  新环境/新角色/新目标
Ch 6-11  [试探]  小循环1完整走完（蓄→爆→收）
Ch 12-18 [受挫]  小循环2（爆发后暴露弱点）+ 大反派正式登场
Ch 19-25 [噩梦]  三向压迫交汇，至暗时刻
Ch 26-30 [爆发]  大逆转 + 奖励结算 + 新卷预告
```

### 1.2 `收束` Phase

`收束` 不是五段式的独立阶段，而是**爆发之后的尾声处理**（通常 1-2 章）。用于：
- 奖励结算（等级跃迁/地位确认/旁人反应）
- 伏笔播种（为下一卷埋线）
- 情绪着陆（从高潮回到日常，给读者喘息）

当卷末最后 1-2 章需要同时完成"爆发收尾 + 新卷预告"时，标记为 `收束`。

### 1.3 变体模板

不同叙事风格可在五段式基础上变形：

| 变体 | 特征 | 适用场景 |
|------|------|---------|
| **串珠式** | 多个独立事件复杂度递增，卷末汇集 | 诡秘之主式多事件卷 |
| **层层递进** | 伏笔逐步引爆，像剥洋葱 | 悬疑/推理卷 |
| **旋涡式** | 涉及面持续扩大，越卷越大 | 格局升级卷 |
| **双线交织** | 两条线交替推进，在高潮合流 | 多 POV 卷 |

变体仍使用 Phase 标注——Phase 描述的是**情绪功能**而非固定结构。例如串珠式中每颗"珠子"内部也有 试探→受挫→小爆发 的微循环。

## §2 蓄力→爆发→收束循环

一卷内通常包含 **2-3 个小循环** 和 **1 个大循环**：

```
卷结构 = [小蓄力→小爆发→短收束] × 2-3 + [大蓄力→大爆发→卷收束]
```

### 2.1 小循环（3-5 章）

单个事件/副本/对手的完整弧线：

| 阶段 | 章数 | 内容 |
|------|------|------|
| 蓄力 | 1-2 | 压迫感建立、信息差制造、配角质疑 |
| 爆发 | 1-2 | 打脸/战斗/揭秘/逆转 |
| 收束 | 0.5-1 | 奖励结算、旁人反应、伏笔播种 |

小循环的收束**同时是下一个循环的蓄力起点**——上一个对手被击败后暴露了更大的威胁。

### 2.2 大循环（整卷）

大循环覆盖整卷，由 Phase 体系驱动：
- `期待` + `试探` = 大蓄力前半段
- `受挫` + `噩梦` = 大蓄力后半段（压力持续升级）
- `爆发` = 大爆发 + 卷收束

### 2.3 循环叠加规则

- 小循环嵌套在大循环的 Phase 内——`试探` Phase 通常包含 1 个完整小循环，`受挫` Phase 包含 1-2 个
- **螺旋上升**：每个小循环的基线必须高于上一个——不是同一层面反复，而是质的提升
- **情绪公式**：`小悲→小喜→小怒→小爽 → 中悲→中喜→中怒→中爽 → 大悲→大喜→大怒→大爽`

## §3 Phase → excitement_type 排列指导

excitement_type 不应随机选择，而应与 Phase 和循环位置配合：

| Phase | 推荐 excitement_type | 不推荐 |
|-------|---------------------|--------|
| `期待` | `worldbuilding_wow`, `setup`, `mystery_reveal` | `power_up`（太早升级贬值） |
| `试探` | `confrontation`, `reversal`（小打脸）, `setup` | `emotional_peak`（情绪释放太早） |
| `受挫` | `cliffhanger`, `setup`, `mystery_reveal` | `power_up`（受挫期不宜升级） |
| `噩梦` | `cliffhanger`, `emotional_peak`（悲向）, `confrontation` | `setup`（已无蓄力空间） |
| `爆发` | `power_up`, `reversal`, `confrontation`, `emotional_peak`（燃向） | `setup`（爆发期禁止铺垫） |
| `收束` | `worldbuilding_wow`（新卷预告）, `cliffhanger`（卷末钩子） | `power_up`（刚爆发完再升级太密） |

### 3.1 序列模式示例（25 章卷）

```
Ch 1  [期待]  worldbuilding_wow     — 新世界/新势力震撼登场
Ch 2  [期待]  setup                 — 目标确立，伏笔播种
Ch 3  [期待]  mystery_reveal        — 关键信息揭示，驱动力建立
Ch 4  [期待]  setup                 — 缓冲，角色关系铺垫
Ch 5  [试探]  confrontation         — 小循环1蓄力：首次对抗
Ch 6  [试探]  setup                 — 蓄力：准备/谋划
Ch 7  [试探]  reversal              — 小循环1爆发：小打脸
Ch 8  [试探]  setup                 — 收束+播种
Ch 9  [试探]  mystery_reveal        — 新信息引出更大威胁
Ch 10 [受挫]  confrontation         — 小循环2蓄力：正面碰壁
Ch 11 [受挫]  cliffhanger           — 形势急转直下
Ch 12 [受挫]  setup                 — 主角试图应对
Ch 13 [受挫]  reversal              — 小循环2爆发：局部胜利但暴露弱点
Ch 14 [受挫]  cliffhanger           — 大反派正式出手
Ch 15 [受挫]  mystery_reveal        — 揭示真正威胁的规模
Ch 16 [噩梦]  confrontation         — 三向压迫开始
Ch 17 [噩梦]  cliffhanger           — 同伴背叛/资源断裂
Ch 18 [噩梦]  emotional_peak        — 情绪谷底
Ch 19 [噩梦]  cliffhanger           — 看似无解
Ch 20 [噩梦]  mystery_reveal        — 关键底牌信息获取
Ch 21 [噩梦]  setup                 — 最后的准备（仅此一章）
Ch 22 [爆发]  reversal              — 大逆转开始
Ch 23 [爆发]  confrontation         — boss 战/终极对决
Ch 24 [爆发]  power_up              — 突破/终极爆发
Ch 25 [收束]  cliffhanger           — 奖励结算 + 卷末悬念
```

### 3.2 关键约束

- **setup 不连续**：连续 2 章以上 setup 会导致读者流失
- **爆发期禁止 setup**：`爆发` Phase 内不应出现 setup（情绪正在释放，不能刹车）
- **噩梦期慎用 power_up**：至暗时刻升级会破坏绝望感
- **每 3-5 章至少 1 个爽点**：reversal / power_up / confrontation 中的胜利

## §4 钩子链编排

### 4.1 强度交替原则

```
强钩子（悬崖式结尾）→ 缓钩子（氛围/伏笔）→ 缓钩子（日常/成长）→ 强钩子
```

- 每章都用强钩子 → 读者疲劳
- 连续 3 章无钩子 → 读者流失
- **1 强 2 缓**是默认节奏，可根据 Phase 加密：`噩梦` 和 `爆发` Phase 可连续 2-3 个强钩子

### 4.2 七种钩子类型

| 钩子类型 | 技法 | 适用场景 |
|---------|------|---------|
| 高压开场钩 | 直接进入情绪高点，无铺垫 | 冲突爆发、反转时刻 |
| 剧情错位钩 | 读者预期 A，叙事转向 B | 节奏放缓段落的尾部 |
| 对话断句钩 | 角色话说一半被打断 | 关键信息揭露前 |
| 动作中断钩 | 物理场景进行到一半戛然而止 | 高强度战斗/追逐 |
| 新设定引入钩 | 章末抛出全新世界观信息 | 新弧线开始前 |
| 情绪高潮钩 | 情感到达顶点但不给结局 | 虐心/感情线 |
| 留白反转钩 | 上章表面结论在下章被颠覆 | 制造剧情反转 |

### 4.3 钩子层级体系

| 层级 | 范围 | 兑现规则 |
|------|------|---------|
| 宏观悬念 | 贯穿全书 | 每 1-2 卷提及一次保持活性 |
| 卷级悬念 | 一卷内 | 卷末必须兑现或明确推进 |
| 章级悬念 | 3-5 章 | **3 章内兑现**（网文读者不等） |
| 段落钩子 | 每个场景 | 即时翻页动力 |

**核心原则**：钩了就必须兑现。超过百章未回收的悬念，读者已经忘了。

## §5 信息差模式

### 5.1 四种信息差

| 类型 | 谁知道更多 | 情绪效果 | 例子 |
|------|-----------|---------|------|
| 读者 > 角色 | 读者知，角色不知 | 期待感/紧张感（"他要倒霉了"） | 读者知药老在，萧炎表面必败 |
| 角色 > 读者 | 角色知，读者不知 | 悬念/好奇心（"他的底牌是什么"） | 克莱恩的愚者身份层层揭示 |
| 角色 A > 角色 B | 角色间不对称 | 期待"掉马甲"时刻 | 身份隐藏与暴露 |
| 全局未知 | 谁都不知道 | 恐惧/未知感 | 真实造物主的本质 |

### 5.2 信息差 × Phase 配合

| Phase | 推荐信息差操作 |
|-------|--------------|
| `期待` | **建立**信息差（角色>读者：抛出谜团；读者>角色：透露威胁） |
| `试探` | **扩大**信息差（更多线索但不揭底） |
| `受挫` | **翻转**部分信息差（读者>角色 → 角色发现威胁；新增角色A>角色B） |
| `噩梦` | **利用**信息差制造绝望感（角色>读者的底牌尚未亮出） |
| `爆发` | **兑现**信息差（底牌亮出、身份揭露、真相大白） |

## §6 承诺-兑现周期

### 6.1 三层期待

| 层级 | 兑现周期 | 卷级对应 |
|------|---------|---------|
| 短期 | 3 章内 | 小循环内的即时满足 |
| 中期 | 1 卷内 | 卷目标达成、小 boss 击败 |
| 长期 | 跨卷 | 终极目标/核心谜团 |

### 6.2 爽点间距

- **短期爽点**：每 3 章一个小打脸/小胜利（维持追读动力）
- **中期爽点**：每 8-12 章一次阶段性大胜（小循环的爆发点）
- **卷级爽点**：卷末集中释放（多条线的压力同时解决）
- **关键**：小高潮之后**必须有缓冲**（1-2 章 setup），否则大高潮没有情绪空间

### 6.3 三向压迫法

在 `噩梦` Phase 的大蓄力阶段，从三个方向同时施压：
1. **敌人/目标**的威胁与质疑
2. **配角/同伴**的不信任或背叛
3. **主角自身**的困境与无力感

三重压迫后的逆转（`爆发` Phase），情绪释放是单向压迫的 3 倍以上。

## §7 迷你卷（黄金三章）的节奏

3 章迷你卷的特殊节奏，不套用五段式，而是：

| 章 | Phase | 重点 |
|----|-------|------|
| 1 | `期待` | 人物出场 + 世界观冲击 + 核心矛盾暗示 |
| 2 | `试探` | 制造冲突 + 第一个小高潮 + 伏笔播种 |
| 3 | `受挫` 或 `爆发` | 确立读者期待方向 + 强钩子（吸引继续追） |

3 章内完成一个**微型承诺-兑现周期**：Ch1 提出问题 → Ch2 尝试解决 → Ch3 部分解决但暴露更大问题。
