# 角色动机设计方法论

本文档为 PlotArchitect 和 ChapterWriter 提供角色动机设计指导。核心目标：让角色的每个选择**令读者信服**，而非服务于剧情需要的工具人。

## 使用方式

- **PlotArchitect**：规划大纲时，为主要角色设计核心驱动力，确保每章冲突与角色动机有因果关联（Process 步骤 2-3）
- **ChapterWriter**：写作时，让角色的行为和选择能追溯到动机——读者不需要被告知"他为什么这样做"，但能从行为中感受到

> 本文档为方法论指导，不要求在 L2/L3 中新增结构化字段——动机通过叙事呈现，不通过标签管理。

---

## §1 核心驱动力（每个角色 1-2 条）

好角色不需要复杂的心理画像，但需要**清晰的驱动力**——读者在任何时刻都能回答"这个角色想要什么"。

### 1.1 驱动力三要素

| 要素 | 定义 | 例子（克莱恩·莫雷蒂） |
|------|------|----------------------|
| **欲望**（Want） | 角色主动追求的目标 | 回到地球、回到原来的生活 |
| **恐惧**（Fear） | 角色回避的后果 | 失控、变成非人的存在 |
| **矛盾**（Contradiction） | 欲望与恐惧的冲突 | 要回家必须升序列，升序列就越来越不像人 |

**矛盾是引擎**——它让角色每次抉择都有内在张力，而不是简单地"打怪→变强→打更强的怪"。

### 1.2 主角 vs 配角的驱动力深度

| 角色级别 | 驱动力要求 | 呈现密度 |
|---------|-----------|---------|
| 主角 | 欲望 + 恐惧 + 矛盾，三者完整 | 每 3-5 章通过选择体现一次 |
| 核心配角 | 欲望 + 1 个明确的行为倾向 | 每次出场有一致的行为逻辑 |
| 功能性角色 | 1 个清晰标签（贪婪/忠诚/野心） | 出场即标签可见 |
| 反派 | 必须有自洽的欲望（不能纯粹"为恶而恶"） | 至少 1 次视角展示动机 |

### 1.3 网文常见驱动力模式

| 模式 | 欲望 | 恐惧 | 矛盾来源 | 典型作品 |
|------|------|------|---------|---------|
| **复仇型** | 向害自己的人复仇 | 在复仇过程中失去更多 | 复仇代价不断升级 | 斗破苍穹 |
| **守护型** | 保护重要的人 | 保护能力不足 | 变强的方式可能伤害被保护者 | 诡秘之主 |
| **探索型** | 解开世界的秘密 | 真相可能比无知更可怕 | 每个答案带来更大的谜团 | 诡秘之主 |
| **崛起型** | 从底层爬到顶端 | 被已有势力碾碎 | 要借助的力量不可控 | 凡人修仙传 |
| **自由型** | 不受约束地活着 | 被体制/命运困住 | 要获得自由必须先获得力量，力量带来责任 | 赤心巡天 |

## §2 动机-抉择链（让选择令人信服）

### 2.1 每个重要选择必须能回答三个问题

1. **角色为什么非做不可？**（与核心驱动力的连接）
2. **为什么是现在？**（外部压力或信息揭示让选择不可推迟）
3. **为什么选这个而不选那个？**（代价对比，两害相权取其轻）

如果某个选择只是因为"剧情需要他去那里"，角色就变成了工具人。修复方法：**倒推动机**——先确定角色的 Want/Fear，再设计一个让他不得不做这个选择的情境。

### 2.2 抉择的四种类型

| 类型 | 特征 | 情绪效果 | 适用 Phase |
|------|------|---------|-----------|
| **两难选择** | A 和 B 都有代价，不存在完美选项 | 纠结、共鸣 | 受挫/噩梦 |
| **牺牲选择** | 为更高目标放弃珍视之物 | 心痛、敬佩 | 噩梦/爆发 |
| **信仰选择** | 在没有足够信息时基于信念行动 | 悬念、代入 | 试探/受挫 |
| **本能选择** | 在极端压力下暴露真实性格 | 震撼、真实感 | 爆发 |

### 2.3 选择的后果必须不可逆

- 好的选择改变世界状态——L3 契约「局势变化」表应该反映选择的后果
- **选完没后果 = 假选择**——读者会感到被欺骗
- 后果应该在后续章节中持续产生涟漪，不是选完就翻篇

## §3 动机演进（角色弧光）

### 3.1 网文角色弧线的四种模式

| 模式 | 轨迹 | 适用角色 | 注意事项 |
|------|------|---------|---------|
| **成长弧** | 弱→强，不自信→坚定 | 主角（最常见） | 成长必须有代价，不能白拿 |
| **堕落弧** | 善→恶，理想→妥协 | 反派前史、配角叛变 | 每一步堕落都有合理动机 |
| **觉醒弧** | 错误认知→真相，执念→放下 | 中后期主角 | 觉醒时刻需要足够的铺垫 |
| **坚守弧** | 始终如一，世界在变他不变 | 配角（信仰型） | 用环境变化凸显坚守的代价 |

### 3.2 动机转化的触发条件

动机不会无缘无故变化。以下事件可以触发动机转化：

- **信息揭示**：发现过去的认知是错误的（"你以为要保护的人，其实是加害者"）
- **失去**：核心欲望的对象被剥夺（保护的人死了 → 动机从守护转为复仇）
- **代价兑现**：之前选择的后果降临（为变强付出的代价终于要承受了）
- **新视角**：通过对手/配角看到自己的另一面

### 3.3 动机转化 × Phase 配合

| Phase | 动机状态 | PlotArchitect 关注点 |
|-------|---------|---------------------|
| `期待` | 动机明确，目标清晰 | 让读者理解"主角想要什么" |
| `试探` | 动机受到初步考验 | 小选择暴露动机的优先级 |
| `受挫` | 动机与现实产生裂痕 | 角色开始怀疑自己的选择 |
| `噩梦` | 动机面临崩塌或被迫转化 | 至暗时刻：要么坚守要么蜕变 |
| `爆发` | 新动机或净化后的旧动机驱动行动 | 角色的选择比开头更坚定/更复杂 |
| `收束` | 新动机确立，为下卷铺垫 | 让读者看到角色成长了什么 |

## §4 动机外化（Show, Don't Tell）

### 4.1 动机通过行为呈现，不通过内心独白解释

| 坏的写法 | 好的写法 |
|---------|---------|
| "他知道自己必须变强来保护苏瑶" | 他看了一眼苏瑶的背影，转身走进了修炼室 |
| "她内心充满了恐惧" | 她的手指在桌下攥紧，指甲陷进肉里 |
| "林枫下定了决心" | "我去。"他没看任何人 |

### 4.2 三种外化手段

| 手段 | 原理 | 适用场景 |
|------|------|---------|
| **选择外化** | 角色在两难中选了哪个，直接暴露优先级 | 抉择时刻 |
| **反应外化** | 对同一事件不同角色反应不同，暴露各自动机 | 群戏/对话 |
| **代价承受** | 角色愿意为目标承受多大代价，量化动机强度 | 噩梦/爆发 Phase |

### 4.3 对话中的动机暗示

对话是动机外化的最高效手段——角色说什么、不说什么、怎么说，都能暗示动机：

- **话题选择**：角色主动聊什么 → 他在乎什么
- **回避反应**：角色避开什么话题 → 他恐惧什么
- **语气变化**：同一角色对不同人的态度差异 → 关系中的权力/情感动态
- **说谎/隐瞒**：角色为什么对这个人撒谎 → 他在保护什么

## §5 多角色动机碰撞

### 5.1 冲突源于动机不兼容

最好的冲突不是"好人 vs 坏人"，而是"两个都有道理的人想要互斥的东西"：

- A 想保护村庄 vs B 想保护国家 → 牺牲村庄保国家？
- A 想揭露真相 vs B 想维持秩序 → 真相会造成更大的混乱？
- A 想复仇 vs C 想和解 → 复仇的对象是 C 的至亲？

### 5.2 反派的动机自洽

反派必须在自己的逻辑里是"对的"——至少他自己认为如此：

- **不要**：纯粹邪恶、为恶而恶、脸谱化反派
- **要**：反派有合理的出发点，只是手段/底线与主角不同
- **最佳**：读者读完反派视角后能理解"如果我是他，可能也会这样做"

### 5.3 配角的动机独立性

配角不应该只为主角服务。每个有名字的配角都应有自己的 Want：

- 当配角的 Want 与主角一致 → 联盟
- 当配角的 Want 与主角冲突 → 紧张/背叛
- 当配角的 Want 改变 → 关系转折点

## §6 常见动机问题清单（ChapterWriter 自检）

写完章节后，对照以下问题检查：

1. **本章主角做了什么选择？** → 如果没有，这一章可能缺乏主角能动性
2. **这个选择能从他的核心驱动力推导出来吗？** → 如果不能，角色可能在当工具人
3. **读者能从行为（不是内心独白）感受到动机吗？** → 如果需要大段解释，外化不足
4. **选择的后果在本章内可见吗？** → 如果选完没有后果，读者会觉得选择无关紧要
5. **配角/反派的行为有自己的逻辑吗？** → 如果对手只是"阻碍主角"的工具，冲突会很假
