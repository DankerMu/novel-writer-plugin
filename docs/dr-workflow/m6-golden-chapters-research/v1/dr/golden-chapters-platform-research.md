> 调研日期：2026-03-01

## 一、概念界定

「黄金三章」是网文圈对**小说前三章（约 6000-10000 字）**决定性作用的共识性表述。核心定义：

> 小说的前三章是决定生死的关键章节，只要在开篇前三章能够抓住读者，那么小说在激烈竞争中活下来的几率就会非常大。

通用建议框架：
- 聚焦主角，少写杂鱼
- 明确风格，突出主线
- 直击爽点，弱化铺垫
- 挑起悬念，引发期待
- 不要有低级错误

但随着平台分化，三大平台对「黄金三章」的要求已出现结构性差异。阅文起点创作学堂 2025 年 5 月发文甚至提出「黄金一章」概念，认为传统黄金三章在当前市场形势下「有些过时和缓慢」。

## 二、番茄小说

### 2.1 平台基因：免费 + 算法推荐

| 属性 | 详情 |
|------|------|
| 商业模式 | 免费阅读 + 广告分成（字节系生态） |
| 日活 | 6000万+（2020 数据），2025 年日活超 1.2 亿 |
| 用户画像 | 18-35 岁下沉市场为主，人均阅读时长 98 分钟/日 |
| 推荐机制 | 纯算法推荐，类抖音逻辑 |
| 签约门槛 | 2 万字可申请（业内最低） |

### 2.2 读者留存机制：翻页率 / 完读率

番茄的流量分配核心公式：

```
流量池 PK 实力 = 总留存 = 投放流量 × 封面点击率 × 内容留存
```

关键指标：
- **总留存**：不是单一完读率，而是所有章节完读总和。两本书十万字完读率相同（30%），但如果 A 书第二章完读 80% 平稳下滑，B 书第二章完读 40% 平稳下滑，B 书总留存远低于 A 书
- **首章完读率门槛**：需 >15%（编辑内部数据：首章留存率低于 30% 的书基本无推荐）
- **第二章追读率**：需 >30%
- **日均阅读人数**：需 >500 人
- **完读曲线**：下降越平缓越好，十万字读完破 20% 机会更大

**对前 3 章的权重极高**：番茄新书的「首秀」（10 万字触发）是平台集中推送 3 周，首秀数据决定作品生死。但前 3 章的完读数据直接决定整体完读曲线的起点高度。

### 2.3 推荐算法对前 3 章的权重

- 签约前纯单机，读者搜不到。签约后 10 万字开始推荐
- 签约时编辑根据评估给作品**初始分数**，前几章质量直接影响初始分
- 系统在「验证期」（7-14 天）重点监测完读率、追读率
- **书名 > 前三章 > 后续内容**：书名决定点击率（封面点击率），前三章决定留存率，两者共同决定总留存

### 2.4 章节字数偏好

- **每章建议 2000-2300 字**（官方建议）——利于有声小说和冲奖励
- 每天码 4000-8000 字，正常发布 2000-6000 字
- 短章策略：碎片化阅读习惯，每章一个小高潮

### 2.5 钩子密度要求

- **每章设置 3 个「情绪节点」**（反转/高潮/悬念）
- 每 300 字埋设一个悬念
- 每章结尾必须设置「悬崖时刻」
- 首章需包含**「主角危机 + 金手指 + 悬念」**三要素
- 开篇 200 字内设置核心冲突

### 2.6 编辑签约审核标准

编辑评估维度：
1. 内容题材是否适合签约
2. 故事情节原创性、连贯性、吸引力
3. 人物塑造鲜明度
4. 文笔流畅度
5. 市场潜力评估

签约机会：2万/5万/8万字各一次，3 次全失败则该作品永不签约。

**前几章重点**：
> 「前几章一定要好好写。签约的时候，编辑会根据评估结果给你的作品一个初始分数。章节发表十天后就不能再修改了。」——番茄作者经验分享

### 2.7 番茄黄金三章公式

```
第 1 章（2000字）：主角登场 → 身处困境/危机 → 金手指初现 → 章末悬念
第 2 章（2000字）：金手指激活 → 首次小爽点 → 新威胁浮现 → 章末钩子
第 3 章（2000字）：首场冲突解决 → 打脸/逆袭 → 更大期待抛出 → 长线悬念
```

**核心词：快、爽、密**

## 三、起点中文网

### 3.1 平台基因：付费 + 编辑推荐 + 数据双轨

| 属性 | 详情 |
|------|------|
| 商业模式 | VIP 付费订阅 + 打赏 + IP 衍生 |
| 历史 | 2002 年创立，2003 年首创 VIP 付费阅读制度 |
| 用户画像 | 25-40 岁，教育程度较高，付费意愿强 |
| 推荐机制 | 编辑推荐 + 数据双轨制（人工 + 智能推） |
| 签约门槛 | 2 万字入库编辑审核，3 万字正式评级（较高） |

### 3.2 收藏/推荐票与前 3 章的关系

起点的核心数据指标体系：
- **新书入库**：签约后满 2 万字，稳定更新 5 天，进入系统流量分发（持续 10 天）
- **试水推荐（1-4 轮）**：每周根据**追读数据**顺位 PK 晋级
- **三江推荐**：同期最强作品进入首页
- **首页强推**：通常在三江期间上架

收藏的意义：
- 新书榜潜力值 = **当周会员点击 + 当周推荐×5 + 总收藏×2**
- 收藏是长期留存的核心指标，但前 3 章的「代入感 + 期待感」直接决定读者是否收藏
- 收订比（收藏人数 / 首订数）反映正版转化率，优秀值约 10:1

### 3.3 设定党读者对世界观展示的期待

起点读者群体中存在大量「设定党」——他们对世界观的精细度、逻辑自洽性有极高要求。

**起点 vs 番茄的核心差异**：
- 番茄读者：「快给我看主角装逼」
- 起点读者：「快给我看这个世界怎么运转的」

起点编辑建议的开头要素：
1. **代入感**：读者能快速代入主角处境
2. **期待感**：故事未来走向值得期待
3. 世界观可以慢慢展开，但必须在前 3 章给出足够的「世界独特性暗示」
4. 金手指不必急于亮相，但主角的困境和核心矛盾必须清晰

经典案例分析：
- 《超级怪兽工厂》：前 3 章无打脸、无女角、金手指未激活，但清晰交代了主角的加工厂困境和人物动机 → 万订
- 《诡秘之主》：开篇自杀悬念，第一章就建立了蒸汽朋克世界的独特氛围 → 均订 10 万

### 3.4 黄金三章 vs 黄金五章

起点圈内有「黄金五章」的说法，原因：

1. **起点读者更有耐心**：付费用户筛选效应，愿意给作品更长的考察期
2. **编辑审核以 2-3 万字为标准**：实际看的是前 5-10 章
3. **10 万字智能推**：作品满 10 万字后进入系统抓取进行流量测试，给前期慢热作品翻盘空间
4. **复杂世界观需要铺垫空间**：硬科幻、仙侠体系无法在 3 章内完成

起点编辑的实际标准：
> 「作品两万字内会进入编辑后台审核，三至七个工作日。有时内容不好判断会再跟进，看后续笔力和控制，审核时间延长至一个月。」——起点 4 组编辑

但核心原则不变：
- 前 3 章建立「我要看下去」的冲动
- 前 5 章完成「世界观骨架 + 核心矛盾 + 主角独特性」
- 前 3 万字决定签约与否

### 3.5 VIP 章节收费模式对开头策略的影响

起点的付费结构：
- 免费期（上架前）：积累收藏和推荐票
- VIP 上架（通常 20-30 万字）：开始收费，首订决定后续推荐力度
- 订阅分成：千字 2-3 分（根据 VIP 等级）

**开头策略影响**：
1. 前 3 章不直接产生收入，但决定收藏量
2. 收藏量决定上架时的首订数
3. 首订数决定后续推荐资源
4. 因此前 3 章的投资回报是**延迟但巨大的**

允许的开头模式：
- **慢热型**：世界观铺垫 → 人物弧光 → 逐步升温（大神专属）
- **标准型**：困境 → 奇遇 → 小爽点 → 期待感（新人推荐）
- **设定先行型**：独特世界规则展示 → 主角在规则中的独特位置 → 悬念

### 3.6 起点黄金三章公式

```
第 1 章（3000-4000字）：主角登场 + 背景暗示 + 核心困境建立 + 世界独特性初露
第 2 章（3000-4000字）：困境深化 + 金手指/奇遇线索 + 人物关系初建
第 3 章（3000-4000字）：首个小转折 + 世界观骨架成型 + 主线期待感建立
```

**核心词：深、稳、期待**

## 四、晋江文学城

### 4.1 平台基因：女频 + 编辑推荐 + 积分榜单

| 属性 | 详情 |
|------|------|
| 商业模式 | VIP 付费订阅 + 打赏 + 影视/出版版权 |
| 创立 | 2003 年，国内最大女性向原创文学平台 |
| 月活 | 535 万（2023.11） |
| 用户画像 | 年轻女性为主，品牌忠诚度高 |
| 推荐机制 | 编辑推荐（人工榜）+ 积分自然排行 |
| 签约门槛 | 1 万字可申签，签人（5 年起步独家） |

### 4.2 榜单机制与前 3 章数据的关系

晋江的榜单体系：

**自然榜单**（积分驱动）：
- 积分 = f(授权年限系数, 字数, 点击, 收藏, 评论数, 评论字数, 读者打分)
- 新晋作者榜：注册作者号后 30 天内的积分排行
- 月度/季度/半年/总分排行榜

**人工榜单**（编辑排期）：
- 编辑推荐榜（鞭腿）：V 前作品，3 万字可上
- 分类封推（红图/红字）：编辑选择的优质作品
- 佳作推荐：入 V 后编辑排榜
- 新书千字金榜（夹子）：入 V 后第 4 天根据前 3 天千字收益排名

**前 3 章对数据的影响**：
- 收藏量：读者看完前几章决定是否收藏，直接影响积分
- 24 小时末章点击：衡量追更活跃度
- 收藏 vs 末点比例：若收藏远高于 24 小时末点（超过 3:2），说明读者已弃文
- 前 7 日互动数据（收藏/评论/打赏）：系统据此决定是否推入「新文推荐」池

### 4.3 女频读者对人设/CP/情感基调的敏感度

晋江读者的核心关注点与男频截然不同：

| 维度 | 晋江读者期待 | 男频（起点/番茄）读者期待 |
|------|------------|---------------------|
| 第一关注 | 人设是否有吸引力 | 金手指/爽点是否给力 |
| 第二关注 | CP 化学反应 | 世界观/升级体系 |
| 第三关注 | 情感基调/氛围感 | 打脸/逆袭节奏 |
| 文笔要求 | 较高，需流畅有质感 | 可接受白话/口水文 |
| 容忍慢热 | 中等（看文笔和人设） | 低（番茄）/中高（起点） |

前 3 章必须做到：
1. **人设立住**：通过言行展示主角多帅/多可爱/多独特，塑造具体性格
2. **CP 信号**：男女主（或攻受）必须在前 3 章内至少一方登场，且给出 CP 期待
3. **风格定调**：轻松风要有笑点，正剧风要抛出疑团或独特设定
4. **萌点元素展现**：星际、萌宠、穿书等标签元素必须在前 3 章体现
5. **禁忌**：别写楔子/前言，别长篇大论介绍背景，别群像开场，别水无意义对话

晋江签约攻略的核心建议：
> 「黄金三章就是你文前面三个章节。开篇节奏要快！做到这些加文笔别太差基本就及格了。」

### 4.4 不同子类型的开头策略

#### 古代言情/古代穿越
- 开篇常见套路：重生/穿越后立即面临危机（如被全族嫌弃、被主母虐待、国破家亡）
- 前 3 章需建立：时代感 + 女主性格反差（前世柔弱 → 今生白切黑）+ 男主初登场或暗示
- 热门元素：宅斗、女强、种田、神医、权谋
- 字数偏好：每章 3000-4000 字

#### 现代言情
- 开篇常见套路：闺蜜背叛/被渣男伤害 → 重生/觉醒 → 遇见真命天子
- 前 3 章需建立：都市感 + 女主独立人设 + CP 初遇的化学反应
- 热门元素：先婚后爱、双强、破镜重圆、契约婚姻
- 语言风格：流畅自然，偶尔俏皮，避免过度堆砌辞藻

#### 穿书/快穿/系统流
- 开篇要素：原书设定快速交代 + 系统/任务亮相 + 主角在新世界的第一个冲突
- 前 3 章需建立：原书与现实的张力 + 金手指规则 + 男主黑化值/好感度等量化期待
- 节奏比古言更快，接近男频的钩子密度

### 4.5 预收与正文开头的衔接策略

晋江独有的「预收」机制是影响开头策略的重要因素：

**预收机制**：
- 作者在当前连载作品的作者公告区放置下一本书的预收文案
- 读者可提前收藏预收作品，积累初始数据
- 预收收藏数直接影响新文开文时的初始曝光

**衔接策略**：
1. **文案与正文一致性**：预收文案承诺的萌点/类型/CP 必须在前 3 章兑现，否则引发退收潮
2. **开文时机**：利用新晋榜（注册作者号后 30 天），提前存稿 1-3 万字一次性发出
3. **预收读者心理**：这批读者对作品有明确预期（已读过文案），前 3 章需快速验证预期
4. **风格延续**：如果是同类型续作，老读者期待熟悉的风格；如果是转型之作，需在文案中明确提示

### 4.6 晋江黄金三章公式

```
第 1 章（3000-4000字）：主角登场 + 核心人设展示 + 情境建立 + 风格定调
第 2 章（3000-4000字）：CP 对手戏/暗示 + 核心冲突启动 + 情感基调确立
第 3 章（3000-4000字）：首个情感转折 + 萌点/标签元素集中展示 + 追更欲建立
```

**核心词：人设、情感、氛围**

## 五、跨平台对比

### 5.1 同一个故事在不同平台的开头策略差异

假设故事核心：「女主重生回到高中，发现自己被闺蜜和渣男联手害死」

| 维度 | 番茄版本 | 起点版本（若有女频频道） | 晋江版本 |
|------|---------|---------------------|----------|
| 第 1 章重点 | 死亡场景 → 重生瞬间 → 系统绑定 | 重生后的内心独白 + 世界设定暗示 | 重生后第一个场景的氛围营造 + 人设展示 |
| 节奏 | 极快，200 字内进入冲突 | 中等，可铺垫 500-1000 字 | 中等偏慢，注重文笔质感 |
| 金手指出场 | 第 1 章末必须出现 | 可延至第 2-3 章 | 可延至第 3-5 章 |
| CP 登场 | 可选，不是核心 | 不是核心关注 | 必须在前 3 章内有暗示 |
| 爽点密度 | 每章 3+ 个情绪节点 | 每 2-3 章 1 个转折 | 每章 1-2 个情感波动 |
| 章节字数 | 2000-2300 字/章 | 3000-4000 字/章 | 3000-4000 字/章 |
| 书名风格 | 「离婚后，首富前夫跪求复婚」 | 「重生之都市仙尊」 | 「重生后我靠摆烂爆红」 |

### 5.2 核心差异总结

| 维度 | 番茄小说 | 起点中文网 | 晋江文学城 |
|------|---------|-----------|------------|
| **核心算法** | 纯算法（完读率驱动） | 编辑 + 数据双轨 | 编辑 + 积分公式 |
| **前 3 章使命** | 最大化完读率和追读率 | 最大化收藏和追读 | 最大化收藏和互动 |
| **读者耐心窗口** | 极窄（300 字定生死） | 中等（3 万字定签约） | 中等偏宽（人设好可慢热） |
| **章节字数** | 2000-2300 字 | 3000-4000 字 | 3000-4000 字 |
| **钩子密度** | 极高（每 300 字 1 个） | 中等（每章 1-2 个） | 中低（情感驱动为主） |
| **世界观展示** | 最小化（别拦着爽点） | 鼓励（设定党需求） | 适度（服务于人设和情感） |
| **人设权重** | 低（功能性即可） | 中（需有记忆点） | 极高（核心卖点） |
| **CP 权重** | 低 | 低 | 极高 |
| **文笔要求** | 低（通顺即可） | 中（需有节奏感） | 较高（需质感和氛围感） |
| **金手指要求** | 必须快速亮相 | 可适当延后 | 看类型，非必须 |
| **允许慢热** | 否（几乎不允许） | 有条件允许（大神/硬设定） | 有条件允许（文笔/人设好） |
| **「黄金 N 章」** | 黄金一章（甚至黄金 300 字） | 黄金三至五章 | 黄金三章（人设定调型） |

### 5.3 各平台编辑/运营的公开建议汇总

#### 番茄官方
- 「前几章一定要好好写，编辑会根据评估给初始分数」
- 「一定要重视书名，书名的影响非常大」
- 书名、简介、标签影响冷启动阶段的 1000-5000 人目标推送
- 更新时间避开 22-24 点（卡审导致数据腰斩）
- 首秀期千万不要断更和大改

#### 阅文起点
- 2025 年提出「黄金一章」概念：「以现在的市场形势再回看黄金三章的要求，似乎有些过时和缓慢了」
- 起点 4 组编辑无书：「稳定更新，稳定更新，稳定更新，重要的事情说三遍」
- 「一个好的稿子，一定是代入感 + 期待感，两者相辅相成」
- 新书期断更 >4 天，取消整本书全勤
- 编辑审核分四级：直接签约 / 重点跟进 / 普通跟进 / 直接放弃

#### 晋江编辑
- 「签约是编辑们综合考量的，暂时无法详细说明」
- 签约后如果能为网站带来利益，晋江会给尽量多的推荐
- 新人建议：利用新晋榜（30 天），提前存稿 1 万字一次性发完
- 等待签约过程中保持日更，日更稳定者脱颖而出
- 「开篇节奏要快！别写楔子/前言，别长篇大论介绍背景，别群像，别水对话」

## 六、对 M6 设计的启示

### 6.1 platform_guide 评估权重建议

基于调研数据，建议 QualityJudge 按平台使用以下权重分布（8 维度，总和 = 1.0）：

| 维度 | 番茄小说 | 起点中文网 | 晋江文学城 | 通用（无平台） |
|------|---------|-----------|------------|---------------|
| plot_tension（情节张力） | **0.20** | 0.15 | 0.10 | 0.125 |
| pacing（节奏） | **0.18** | 0.12 | 0.10 | 0.125 |
| character_depth（人物深度） | 0.08 | 0.12 | **0.20** | 0.125 |
| worldbuilding（世界观） | 0.05 | **0.18** | 0.10 | 0.125 |
| emotional_resonance（情感共鸣） | 0.10 | 0.08 | **0.18** | 0.125 |
| prose_quality（文笔质量） | 0.07 | 0.12 | **0.15** | 0.125 |
| hook_and_suspense（钩子悬念） | **0.22** | 0.13 | 0.10 | 0.125 |
| coherence（连贯性） | 0.10 | 0.10 | 0.07 | 0.125 |

**设计依据**：
- 番茄：hook_and_suspense 和 plot_tension 占 42%，因为算法完读率驱动
- 起点：worldbuilding 独占 18%，反映设定党需求
- 晋江：character_depth + emotional_resonance 占 38%，反映人设和情感核心地位

### 6.2 Step F0 迷你卷规划的平台差异化建议

| 参数 | 番茄小说 | 起点中文网 | 晋江文学城 |
|------|---------|-----------|------------|
| 章节字数 | 2000-2300 字 | 3000-4000 字 | 3000-4000 字 |
| 前 3 章总字数 | 6000-7000 字 | 9000-12000 字 | 9000-12000 字 |
| 钩子密度 | 每 300 字 1 个 | 每 1000 字 1 个 | 每 1500 字 1 个 |
| 金手指出场 | 第 1 章末 | 第 2-3 章 | 视类型，非强制 |
| CP 首次互动 | 不强制 | 不强制 | 前 3 章内必须 |
| 世界观展示量 | 最小（仅服务剧情） | 中大（可独立段落） | 适中（服务人设） |
| excitement_type | 爽点密集型 | 期待构建型 | 情感沉浸型 |

### 6.3 黄金三章的关键质量门 (Gate) 建议

为 QualityJudge 新增的平台感知评分中，建议对前 3 章设置额外硬门：

**番茄硬门**：
- 主角必须在第 1 章 300 字内登场
- 第 1 章必须包含至少 1 个冲突事件
- 每章末必须有明确的悬念钩子
- 3 章内必须完成 1 次「逆袭/打脸/爽点」

**起点硬门**：
- 主角必须在第 1 章 1000 字内登场
- 前 3 章必须建立世界观的基础骨架
- 前 3 章必须明确核心矛盾/困境
- 代入感评分不得低于 3.5

**晋江硬门**：
- 主角人设必须在前 2 章内通过行为展现（非旁白描述）
- CP 至少一方必须在前 3 章登场
- 情感基调必须在前 2 章确立
- 文笔质量评分不得低于 3.5
